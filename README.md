## Compiling 
### Compile with: 
```
make 
```
### System Requirements 
- Mac OS 10.12 + 
- OpenGL

## Running 
```
./plyviewer plymodels/beethoven.ply
```

## Viewing 
- W/S keys used to move camera up and down
- A/D keys to zoom camera in and out
